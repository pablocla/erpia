# Parametrización y Cascada de Features

El ERP Argentina permite activar o desactivar módulos enteros y modificar la cascada de comportamiento de las características del sistema de manera declarativa.

---

## ⚙️ Acceso a la Cascada de Features

Para ver y configurar la matriz interactiva de parametrización (Capa 1 y Capa 3), accede directamente al módulo de control:

👉 [Configurar Cascada de Features & Workflows](/dashboard/capacitacion/parametrizacion)

---

## 💡 Guía de Modificación
1. **Activación de Features**: Permite prender el sistema de trazabilidad de stock, el motor de cálculo de comisiones, o integraciones impositivas adicionales.
2. **Workflows del Sistema**: Modifica los disparadores del Railroad Engine (flujo automatizado de venta a cobro).
