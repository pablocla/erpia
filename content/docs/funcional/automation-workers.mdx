# Empleados Virtuales (Virtual Workers)

Los **empleados virtuales** son perfiles lógicos que ejecutan playbooks NOP en horarios programados, sin intervención humana. Complementan n8n cuando no hay conectividad saliente o como fallback local.

---

|--------|-----|------|-----------|
| Ana Reposición | `deposito` | `0 8 * * 1-5` | `stock_guardian`, `procurement_guardian_n8n` |

Ver el catálogo completo en `/dashboard/automatizacion` o `GET /api/automation/catalog`.

---

## Ejecución

1. El cron `GET /api/cron/automation-virtual-workers` (protegido con `CRON_SECRET`) recorre workers activos.
2. Cada worker dispara sus playbooks vía `lib/automation/playbook-runner.ts`.
3. Los playbooks pueden emitir eventos NOP (`emitToN8n`) o crear tareas/usuarios virtuales.

---

## Parametrización

Desde la UI o `PUT /api/automation/config`:

```json
{
  "activo": true,
  "metadata": {
    "modoConexion": "webhook"
  }
}
```

Valores de `modoConexion`: `webhook` (default), `poll`, `both`.
