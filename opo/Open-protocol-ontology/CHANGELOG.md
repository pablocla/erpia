# OPO Changelog

All notable changes to the Open Protocol ONTOLOGY (OPO) specification will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and the protocol adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [0.1.0] - 2025-06-12

### Added
- Created the core specification containing **6 Protocol Invariants** (OPO-INV-001 through 006) for machine-to-machine integrations.
- Defined **23 core OPO entities** covering financials, supply chain, core organization, and project modules.
- Published draft-2020-12 valid JSON schemas for all 23 entities with built-in multi-language alias mappings (`x-opo-aliases`) and direct ERP system reference points.
- Established the `/public/opo-context.jsonld` semantic JSON-LD linked data dictionary.
- Created the `/public/opo-registry.json` database of 200+ multi-language terms and confidence weight factors.
- Wrote the `/public/opo-ai-primer.md` standard instruction set for prompt injection in AI workflows.
- Completed functional example manifests mapping SAP, TOTVS/Protheus, and Odoo systems.
- Developed the web-based visual schema explorer, dynamic spec lookup, and live registry queries portal.
